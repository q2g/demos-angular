About da Vinci

davinci.js is a framework to build visual elments for the qlik sense plattform. The most important design principal is that you can use the visual element easy as qlik extension in the qlik sense enterprise, but also to use it in own angular websites with just Enigma.JS and without the whole qlik.js stack. You can visit us under https://github.com/q2g. The framework ist open source

The rest design principals:

  • accessibility, screenreader ready and keybord usage / shortcuts
  • ready for versioning, possibility to run different versions of the plugin or elements in one page
  • stay close to the qlik ux

The plan is in the end to get even angular 1.x and angular 4+ elements out of one codebase.
